Data Connections

How to read this graph:

File Node: A Google Sheet. Click to open.
Tab Node: A tab within a sheet.
Unknown Source: An imported file that could not be found or accessed.

  • Dashed Lines: Show which tabs "belong" to which file.
  • Solid Arrows: Show data flow. An arrow from a source (Tab A) to a destination (Tab B) means Tab B imports data from Tab A.
  • Hover: Hover over any node to see details.
  • Move: Click and drag nodes (or the background) to reposition.
  • Zoom: Use the + / - buttons or your scroll wheel to zoom.
  • Download: Use the "Download Data" button to get the raw CSV data used for this graph.
